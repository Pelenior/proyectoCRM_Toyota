# proyectoCRM_Toyota
Proyecto creado por Diego Baudot, Aitana Herranz y Pelayo Santos

CRM para una rama ficticia de toyota que se encarga del alquiler de chofers.

## COMO INICIAR EL PROYECTO
1- Descargar dependencias desde la carpeta "proyectoCrm", usando el comando "npm install"
2- Desplegar la página web en el puerto usando el comando "ng serve" en la misma carpeta que el paso anterior
3- Abrir XAMPP y iniciar apache y MySQL. Crear una base de datos llamada "crm_toyota" si no existe ya, e importar ahí el archivo SQL localizado en la carpeta "database"
4- Importar el proyecto de Spring localizado en la carpeta "ApiRestCRM" a Spring Tool Suite o aplicaciones similares. Esperar a que se instalen todas las dependencias
5- Iniciar el archivo en: src/main/java/com/crm/api "ApiRestCrmApplication.java" haciendo click derecho y arrancandolo como "Spring Tool App"
